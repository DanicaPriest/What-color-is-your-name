The short answer:

HTML uses what are called hex codes to identify colors. The hex codes have a mixture of numbers and the letters A-F. If there is a letter not recognized, HTML will convert it to a 0 and that's how your name is seen as a color.

HTML also has 140 color names so if you type 'white' or 'SeaGreen' you will get those colors.
